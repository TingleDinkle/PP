# Parallax Protocol (Window into the Wired)

A cybersecurity visualization tool inspired by "Serial Experiments Lain", featuring a head-tracking "holographic" display.

## Prerequisites

- Python 3.7+
- A webcam

## Setup

1.  **Create a virtual environment:**
    ```bash
    python -m venv venv
    ```

2.  **Activate the virtual environment:**
    - Windows: `venv\Scripts\activate`
    - Mac/Linux: `source venv/bin/activate`

3.  **Install dependencies:**
    ```bash
    pip install pygame opencv-python mediapipe scapy numpy
    ```

## Usage

Run the main script:

```bash
python main.py
```

## Controls

- **ESC**: Quit the application.
